# 🧠 BOB Skills – Hermes Agent Custom Skills

Skills, die wir gemeinsam für Hermes Agent erstellt haben.

| Skill | Beschreibung | Dateien |
|-------|-------------|---------|
| **discord-bot** | Discord Bot Management – Standalone Bots deployen, SOUL.md-Persönlichkeit, OpenRouter-KI, keine hartcodierten Trigger | SKILL.md, Script, Vorlage, Referenzen |
| **guppy-agent** | Guppy-Persönlichkeit für Recherche-Agent (BOB's Klon) | SKILL.md |
| **obsidian** | Obsidian Vault mit iCloud Sync via pyicloud + Syncthing | SKILL.md, 3 Sync-Scripts, 3 Referenzen |

**Einrichtung:** Einfach den gewünschten Skill-Ordner nach `~/.hermes/skills/<kategorie>/` kopieren.
